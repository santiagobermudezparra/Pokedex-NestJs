# Run in Dev

1. Clone the Repo
2. Run
```
  yarn install 

```
3. Install Nest CLI
```
  npm i -g @nestjs/cli

```
4. Run the DB
```
  docker-compose up -d
```

## Stack 
* MongoDB
* Nest
